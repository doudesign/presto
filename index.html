Distributed SQL Query Engine for Big Data

Get Started

What is Presto?

Presto is a open source, distributed SQL query engine optimized for ad-hoc analysis at interactive speed. It supports standard ANSI SQL, including complex queries, aggregations, joins, and window functions. These SQL queries are parsed, optimized, and distributed to workers that have been designed to execute queries against Petabytes of information in seconds or minutes.

What can it do?

Presto queries data where it currently resides and the current version of Presto supports Hive, HBase, relational databases or even proprietary data stores. A single Presto query can combine data from multiple sources, allowing for analytics across your entire organization.

Who is it for?

Presto is targeted at analysts who expect response times ranging from sub-second to minutes. Presto breaks the false choice between having fast analytics using an expensive commercial solution or using a slow "free" solution that requires excessive hardware. If you are currently executing queries with Hive, Presto's efficient approach to turning SQL into an efficient distributed execution plan will yield significant advantages over Hive's approach.

Who uses it?

Facebook uses Presto for interactive queries against several internal data stores, including their 300PB data warehouse. Over 1,000 Facebook employees use Presto daily to run more than 30,000 queries that in total scan over a petabyte each per day. Facebook has released Presto as an open source project under the Apache Software License Version 2, and we look forward to hearing about your use cases and how Presto can help with your interactive analysis.
